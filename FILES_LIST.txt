   ✅ Открыть @BotFather в Telegram
   ✅ Команда /newbot
   ✅ Следовать инструкциям
   ✅ Получить BOT_TOKEN

4. РАЗВЕРТЫВАНИЕ НА RENDER (10 минут)
   ✅ Залить проект на GitHub
   ✅ Создать Web Service на render.com
   ✅ Добавить переменные окружения
   ✅ Бот работает 24/7!
